# contacts-list

From a contacts list database, you can view all contacts, enter a new contact or delete a contact by entering a specified command.

<li>To view all contacts, enter "V" will display each contact in a set of parenthesis with all associated data separated by commas
<li>To enter a new contact, enter "E", and then enter the contact's first name, last name, phone number and email address
<li>To delete a contact, enter "D", and enter the contact's first name and last name to delete the contact associated with that name

This project uses Python and a MySQL connector to connect with a database which stores the contacts and is queried through user input.

Keep track of your contacts with a few simple commands!
